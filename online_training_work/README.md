# Online Training Work Bundle

This folder packages the online-policy training work in one place for review and handoff.

## Contents

- `src/`: online training modules kept only where not already present in the top-level `src/`
- `scripts/`: secure run helper scripts kept only where not already present in top-level `scripts/`
- `results/`: online sweep outputs and trajectory CSVs
- `figures/`: online evaluation figures
- `docs/`: short writeups and architecture notes
